# Modern World - ربات تلگرام استراتژیک جهان مدرن

ربات تلگرام کاملاً دکمه‌ای (بدون دستور متنی) با UI/UX شیشه‌ای. جنگ، اقتصاد، دیپلماسی و سازمان ملل با هوش مصنوعی.

## ✨ ویژگی‌ها

- **بدون دستور متنی** - همه چیز با دکمه‌های شیشه‌ای (💎🧊)
- **ثبت‌نام خودکار** با کشور تصادفی از ۱۰ کشور
- **فروشگاه دکمه‌ای** برای خرید/فروش منابع
- **اعلان جنگ با AI** - هوش مصنوعی دلایل جنگ را بررسی می‌کند
- **نبرد نوبتی** با تاکتیک‌های مختلف
- **سازمان ملل** با نظرسنجی خودکار توسط AI
- **راهنمای تعاملی** درون برنامه
- **دیتابیس SQLite** فایل‌محور
- **لاگ تمام عملیات** در دیتابیس

## 🚀 نصب و اجرا

### پیش‌نیازها
- Node.js 18+
- npm

### نصب محلی

```bash
git clone https://github.com/yourusername/modern-world-bot.git
cd modern-world-bot
npm install
cp .env.example .env
```

### تنظیم `.env`

| متغیر | توضیح | اجباری |
|-------|-------|--------|
| `BOT_TOKEN` | توکن ربات تلگرام از @BotFather | ✅ |
| `AI_API_KEY` | کلید API برای freemodel.dev | ✅ |
| `AI_API_URL` | آدرس API (پیش‌فرض: https://freemodel.dev/v1/chat/completions) | ❌ |
| `GROUP_ID` | آیدی گروه تلگرام برای اعلان‌ها | ❌ |
| `TOPIC_WAR_ID` | آیدی تاپیک جنگ‌ها در گروه | ❌ |
| `TOPIC_UN_ID` | آیدی تاپیک سازمان ملل در گروه | ❌ |

### اجرا

```bash
npm start
```

## ☁️ استقرار در Railway

1. پروژه را به GitHub پوش کنید
2. در [Railway](https://railway.app) پروژه جدید بسازید
3. ریپازیتوری را به Railway متصل کنید
4. متغیرهای محیطی را در Railway تنظیم کنید
5. Railway به صورت خودکار `npm start` را اجرا می‌کند

### متغیرهای محیطی در Railway

| متغیر | مقدار |
|-------|-------|
| `BOT_TOKEN` | توکن ربات شما |
| `AI_API_KEY` | کلید freemodel.dev |
| `RAILWAY_ENVIRONMENT` | `true` |

## 🎮 راهنمای بازی

### ثبت‌نام
- دکمه `🚀 شروع بازی` را بزنید
- یک کشور تصادفی دریافت می‌کنید
- دارایی‌های اولیه: ۱۰۰۰ طلا، ۱۰۰⚔️، ۱۰۰🏭، منابع: ۵۰ 🛢⚙️🌾

### فروشگاه
- خرید قدرت نظامی (+۱۰) → ۱۰۰ طلا
- خرید قدرت اقتصادی (+۱۰) → ۱۰۰ طلا
- خرید نفت/فولاد/غذا (+۲۰) → ۵۰ طلا
- فروش منابع: هر ۱۰ واحد → ۳۰ طلا

### جنگ
1. از منو `⚔️ اعلان جنگ جدید` را بزنید
2. کشور هدف را انتخاب کنید
3. دلیل جنگ را تایپ کنید
4. هوش مصنوعی دلیل را بررسی می‌کند
5. در صورت تأیید، نبرد شروع می‌شود

### تاکتیک‌های نبرد
| اقدام | توضیح |
|-------|-------|
| 💥 حمله سنگین | قدرت بالا، دقت کم |
| 🎯 حمله دقیق | دقت بالا |
| 🛡️ دفاع کامل | کاهش تلفات |
| 🔄 ضدحمله | ریسک بالا، پاداش بالا |

### سازمان ملل
پس از هر راند، AI ۳ لایحه پیشنهادی تولید می‌کند و در تاپیک سازمان ملل ارسال می‌شود.
